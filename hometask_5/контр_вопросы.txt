1. Какие методы используются для работы с итерируемыеми объектами и с итераторами?
Для работы с итерируемыми объектами используются методы __iter__, который возвращает
итератор, и __getitem__, который позволяет получать элементы по индексу. При работе с
итераторами используется метод __next__, возвращающий следующий элемент, если он есть, или 
StopIteration, если его нет. Также у каждого итератора есть метод __iter__. Этот
метод возвращает сам итератор.
2. Чему должен удовлетворять объект, чтобы быть итерируемым?
Итерируемый объект - объект, который способен возвращать элементы по одному, необходимо
наличие метода __getitem__.
3. Чем генератор отличается от итератора?
Генераторы - это класс функций, которые возвращают <generator object>(работающий как
итератор, отдающий элементы по одному).
При этом элементы возвращаются по запросу, затем генератор приостанавливается до следующего
запроса и сохраняет своё состояние.
4. Какое ключевое слово позволяет нам создать функцию-генератор?
Yield. В отличие от return, yield только приостанавливает выполнение тела функции.
